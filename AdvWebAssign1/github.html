GitHub is a Web-based revision control hosting service for software development and code sharing. GitHub offers free public repositories and collaborations for open-source software creation and charges small and large businesses for private repositories and collaboration. From a social perspective, GitHub allows users to change, adapt and improve software — each project is called a "fork" — in its public repositories. Users can work together in teams or alone. Public users create profiles which display repositories and public activity and help programmers find projects.
